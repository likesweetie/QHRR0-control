# AGENTS.md

## Project guidance

This repository contains a robotics CAN dashboard project.

## UI guidance

For any frontend, dashboard, GUI, web UI, CSS, layout, or component styling task:

1. Read `DESIGN.md` before modifying UI code.
2. Treat `DESIGN.md` as the visual design reference.
3. Follow its color, typography, spacing, component, layout, and responsive behavior guidance.
4. Do not copy Notion logos, trademarks, or brand assets.
5. Use the design only as visual inspiration for this project's own CAN dashboard UI.

## Implementation expectations

- Keep UI code modular.
- Separate CAN communication logic from presentation components.
- Prefer a responsive web dashboard layout.
- Do not change backend protocol or CAN message definitions unless explicitly requested.
- After changes, explain which files were modified and how to run the dashboard.
